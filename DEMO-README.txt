==========

To view a demo of the project, please download the following archive and unzip
it inside the `data/` folder:

  https://drive.google.com/open?id=1t78ZC9vEUOBj5gDyg9efxpATznlVgkvD

Once this is done, you can simply run all of the scripts mentioned in the User
Manual (inside `README.md` or `README.pdf`) to see different parts of the
project in action.

The archive contains parts of the data set for the placenta phantom, including
relevant CT scans, segmented meshes, calibration videos, stereo endoscope
footage and records of da Vinci kinematics.

The code is also available at the following GitHub repository:

  https://github.com/TimboKZ/srv-dataset-tools

If you need additional data sets or footage, please contact Timur Kuzhagaliyev
at one of the following email addresses:
  
  zcabtku@ucl.ac.uk
  tim.kuzh@gmail.com
